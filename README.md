# Robot Vision XR V8

Proyecto modular para GitHub Pages.

## Publicación
Sube el contenido completo de esta carpeta a la raíz del repositorio:
- index.html
- css/
- js/

No abras index.html como archivo local. Usa la URL HTTPS de GitHub Pages.

## Prueba
1. Escanea una pared durante 8 segundos.
2. Activa SUELO ON.
3. Escanea piso y escalones.
4. Cambia BOTH → PLANES → POINTS.
